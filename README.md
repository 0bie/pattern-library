## A collection of reusable components

- Install:
  - Launch terminal
  - `git clone https://github.com/0bie/_Blank.git`
  - `cd _Blank && npm install`

- Dev Server: `npm start`

  - URL: [localhost:8080](http://localhost:8080)

- Lint: `npm run lint`

- Build: `npm run build`

- See `package.json`
